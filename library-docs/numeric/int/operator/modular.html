Modular arithmetic operators.

template <typename T, typename U, auto M = (1 << bitsizeof(T))>
inline auto add(T a, U b) §source

Returns (a + b) mod M. a and b must be less than M.

Parameters

  • typename T

    Type of the first operand. Must be unsigned.

  • typename U

    Type of the second operand. Must be unsigned.
